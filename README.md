# Momentum AI

An AI-powered executive productivity platform built using React, Node.js, Firebase, Firestore and Google's Gemini API.

## Features

- AI Goal Planning
- Smart Task Prioritization
- Intelligent Scheduling
- AI Productivity Coach
- Analytics Dashboard
- Firebase Authentication
- Google Cloud Deployment

## Tech Stack

- React
- Node.js
- Express
- Firebase
- Firestore
- Gemini API
- Google Cloud
